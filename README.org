* 功能介绍
平台：Windows

这是一个自写自用的健康管理工具。使用电脑工作经常会忘记休息、忘记喝水……。该工具让您使用电脑一段时间（40分钟），休息一会（2分钟）。休息时， *强制禁用* 鼠标、键盘、CTRL+ALT+DEL等等，只能关机或休息！别抓狂，休息一会吧。

身体是革命的本钱！
